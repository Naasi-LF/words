# Words Study 背单词应用

一款轻量级的词汇学习应用，帮助您高效地学习和记忆新单词。

## 关于

Words Study 是一款简单而强大的词汇学习工具。应用允许您创建多个单词批次，每个批次包含一系列单词及其释义。凭借简洁的界面和直观的功能，Words Study 使学习新单词的过程变得简单有效。

## 功能特点

- 创建多个批次来组织您的词汇学习
- 在每个批次中添加、编辑和删除单词
- 学习模式下默认隐藏单词释义以测试记忆
- 单击即可显示/隐藏单词释义
- 简单直观的用户界面
- 优化的字体提升阅读体验

## 使用方法

1. 主屏幕以卡片形式显示所有批次
2. 点击 + 按钮创建新批次
3. 点击批次卡片开始学习该批次中的单词
4. 点击批次卡片上的编辑图标管理该批次中的单词
5. 在学习模式下，点击单词可显示或隐藏其释义
6. 使用底部的可见性按钮一次性显示或隐藏所有释义

## 技术细节

Words Study 使用 Flutter 构建，采用 SQLite 进行本地数据存储。应用遵循 Model-View-Controller (MVC) 模式，确保代码组织清晰且易于维护。
